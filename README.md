
# Employee Management System

A Java-based Employee Management System developed using Core Java, JDBC, and MySQL.

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee
- Search Employee by ID

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse

## Database
MySQL

## How to Run
1. Create the database using `database.sql`.
2. Update MySQL username and password in `DBConnection.java`.
3. Add the MySQL Connector/J JAR to the project.
4. Run `Main.java`.
